## Quick start

```bash
npx letsrunit init
```

Run this in your project root. `init` is interactive and sets up what you need.

#### What `init` sets up

* Letsrunit CLI
* Cucumber wiring and support files
* Playwright Chromium browser
* GitHub Actions workflow (optional)
* MCP server (optional)
* AI agent skill (optional)
